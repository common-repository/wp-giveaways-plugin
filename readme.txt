== Description ==

This plugin is pretty simple but it can evolve very powerful. It displays current blog entries categorized as Giveaways and allows for setting their time of validity during which they should display on the sidebar widget.

First version, more coming soon.

== Installation ==

1. Upload folder wp-giveaways to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress backend
3. Drag and drop widget Giveaways to the location you want. That is it!
